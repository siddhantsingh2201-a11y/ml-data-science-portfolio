# Data Science Portfolio

This repository contains end-to-end data science and machine learning projects.
Each project includes problem formulation, data cleaning, exploratory data analysis,
feature engineering, model building, and evaluation.

## Projects
- **Customer Churn Prediction**
  - Classification models (Logistic Regression, Random Forest, XGBoost)
  - Feature engineering and model evaluation
- **Sales Forecasting**
  - Time series analysis using ARIMA and Prophet
  - Forecast accuracy comparison

## Tools & Technologies
- Python, Pandas, NumPy, Scikit-learn
- Matplotlib, Seaborn
- Jupyter Notebook
- Git & GitHub
